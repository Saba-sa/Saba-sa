## About

Frontend and blockchain developer specializing in Solidity smart contracts, DeFi/ERC-20 development, and Web3 frontend integration. I build production applications end-to-end — from contract design and deployment to the React/Next.js interfaces that connect to them.

Currently building **SmartStay**, a blockchain-powered rental platform, and taking on freelance blockchain and frontend engagements.
